# 视觉设计和辅助功能

### Dynamic Type动态字体

允许用户自己调整App中的字体大小，有四个原则需要遵守

- 让字体尽可能灵活，可以使用动态字体就多使用动态字体
- 屏幕多宽就用多宽
- 不要因为字体变大而截断内容
- 避免UI看起来头重脚轻

Xcode会提供预览

### Reduce Motion 减少动态效果

大约有1/3的人不习惯动态效果，所以如果可以的话可以读取iOS系统提供的那种不使用动态效果的选项，去适配那个选项

视频自动播放可能也会引发某些人的焦虑，所以如果可以的话也可以读区iOS系统提供的那种不自动播放的选项，去适配那个选项

### Differentiate Without Color 无需用颜色区分

查看用户是否打开了这个色盲选项，如果打开了，就尝试适配，比如不再用颜色区分而是使用图案区分。

如果可以的话，压根不要用仅颜色区分的情况，而是总是提供多种区分方式
